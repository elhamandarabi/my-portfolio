What is JavaScript?

JavaScript is the programming language of the web. While HTML builds structure and CSS adds style, JavaScript makes pages interactive — buttons that do things, forms that validate, content that updates without refreshing.

Your first JavaScript

JavaScript
// This is a comment — JavaScript ignores it

// Show a message in the browser
console.log("Hello, World!");

// Show a popup
alert("Welcome to JavaScript!");

// Store a value
let name = "Elham";
console.log("My name is " + name);

Where to write JavaScript

  • Inside a <script> tag in your HTML file
  • In a separate .js file linked with <script src="app.js"></script>
  • In the browser Console tab (great for testing)
💡 Tip: Open your browser DevTools (F12 or right-click → Inspect → Console) and type console.log("Hello!") — this is where JavaScript developers spend a lot of time debugging!

Variables & Data Types

Variables store data. In modern JavaScript, use let for values that change and const for values that stay the same.

JavaScript
// let — can be reassigned
let age = 22;
age = 23; // ✅ allowed

// const — cannot be reassigned
const country = "Afghanistan";
// country = "USA"; ❌ Error!

// Data types
let name    = "Elham";       // String
let score   = 95;           // Number
let passed  = true;         // Boolean
let nothing = null;         // Null
let unknown;                 // Undefined

String tricks

JavaScript
const first = "Elham";
const last  = "Andarabi";

// Concatenation
console.log(first + " " + last);

// Template literals (modern, preferred)
console.log(`My name is ${first} ${last}`);
💡 Tip: Always use const by default. Only switch to let when you know the value will change. This prevents accidental bugs.

Lesson 3 of 10

Functions

A function is a reusable block of code. You define it once and call it whenever you need it.

JavaScript
// Define a function
function greet(name) {
  return `Hello, ${name}!`;
}

// Call it
console.log(greet("Elham")); // Hello, Elham!
console.log(greet("Sara"));  // Hello, Sara!

// Arrow function (modern syntax)
const add = (a, b) => a + b;
console.log(add(3, 5)); // 8

Default Parameters

JavaScript
function welcome(name = "Guest") {
  return `Welcome, ${name}!`;
}

console.log(welcome());          // Welcome, Guest!
console.log(welcome("Elham")); // Welcome, Elham!
💡 Tip: Arrow functions () => {} are the modern way to write functions. They're shorter and handle this differently — you'll use them constantly in React!

Conditionals & Loops

if / else

JavaScript
const score = 75;

if (score >= 90) {
  console.log("A grade");
} else if (score >= 70) {
  console.log("B grade"); // ← this runs
} else {
  console.log("Keep trying");
}

for loop

JavaScript
// Count from 1 to 5
for (let i = 1; i <= 5; i++) {
  console.log("Number: " + i);
}

// Loop through an array
const fruits = ["apple", "banana", "mango"];
fruits.forEach(fruit => {
  console.log(fruit);
});
💡 Tip: Use === (triple equals) for comparison in JavaScript, not ==. Triple equals checks both value AND type, preventing subtle bugs.

Lesson 5 of 10

Arrays & Objects

Arrays

Arrays store lists of values.

JavaScript
const skills = ["HTML", "CSS", "JavaScript"];

console.log(skills[0]);        // "HTML"
console.log(skills.length);    // 3

skills.push("React");          // add to end
skills.pop();                   // remove last

// map — transform each item
const upper = skills.map(s => s.toUpperCase());
console.log(upper); // ["HTML","CSS","JAVASCRIPT"]

Objects

Objects store key-value pairs — like a profile card.

JavaScript
const student = {
  name: "Elham",
  age:  22,
  skills: ["JS", "Python"],
  greet() {
    return `Hi, I'm ${this.name}`;
  }
};

console.log(student.name);    // "Elham"
console.log(student.greet()); // "Hi, I'm Elham"
💡 Tip: Arrays and Objects are the most important data structures in JavaScript. Master these and you can work with any API or database data!

DOM Manipulation

The DOM (Document Object Model) is how JavaScript sees and controls your HTML. You can find elements, change their content, and respond to user actions.

JavaScript
// Find elements
const title  = document.getElementById("title");
const btn    = document.querySelector(".my-btn");
const all    = document.querySelectorAll("p");

// Change content
title.textContent = "New Title!";
title.style.color  = "blue";

// Add/remove CSS classes
btn.classList.add("active");
btn.classList.remove("hidden");
btn.classList.toggle("open");

Creating Elements

JavaScript
const card = document.createElement("div");
card.textContent = "New card!";
card.className   = "card";
document.body.appendChild(card);
💡 Tip: querySelector uses the same selectors as CSS — so .class, #id, tag all work. Once you know CSS selectors, you know DOM selectors!

Lesson 7 of 10

Events

Events are things that happen — a click, a key press, a form submit. JavaScript lets you listen for events and run code when they happen.

JavaScript
const btn = document.getElementById("myBtn");

// Listen for a click
btn.addEventListener("click", () => {
  alert("Button clicked!");
});

// Listen for keyboard input
const input = document.getElementById("search");
input.addEventListener("input", (e) => {
  console.log("Typing: " + e.target.value);
});

// Prevent form from refreshing page
form.addEventListener("submit", (e) => {
  e.preventDefault();
  console.log("Form submitted!");
});

Common Events

  • click — user clicks an element
  • input — user types in an input
  • submit — user submits a form
  • keydown — user presses a key
  • mouseover — user hovers over element
💡 Tip: Always use addEventListener instead of inline onclick="..." in HTML. It's cleaner and lets you add multiple listeners to the same element.

Fetch API & JSON

The Fetch API lets your JavaScript talk to servers — loading data from APIs without refreshing the page. Data is usually returned as JSON.

JavaScript
// Fetch data from a public API
fetch("https://api.github.com/users/elhamandarabi")
  .then(response => response.json())
  .then(data => {
    console.log(data.name);
    console.log(data.public_repos);
  })
  .catch(err => console.log("Error:", err));

// Modern async/await version (cleaner)
async function getUser() {
  const res  = await fetch("https://api.github.com/users/elhamandarabi");
  const data = await res.json();
  console.log(data);
}
getUser();
💡 Tip: Always use async/await — it looks like normal code and is much easier to read than chained .then() calls. Wrap it in try/catch for error handling.

localStorage & ES6+ Features

localStorage

localStorage lets you save data in the browser — it persists even after closing the tab.

JavaScript
// Save data
localStorage.setItem("username", "Elham");

// Read data
const name = localStorage.getItem("username"); // "Elham"

// Save an object (must stringify)
const user = { name: "Elham", score: 95 };
localStorage.setItem("user", JSON.stringify(user));

// Read it back
const saved = JSON.parse(localStorage.getItem("user"));

Useful ES6+ Features

JavaScript
// Destructuring
const { name, age } = student;

// Spread operator
const newSkills = [...skills, "React"];

// Optional chaining (avoid errors)
const city = user?.address?.city;

// Nullish coalescing
const display = name ?? "Anonymous";
💡 Tip: Destructuring and the spread operator are used constantly in React. Getting comfortable with them now will make learning React much smoother!

Build an Interactive To-Do App

Let's put it all together — a fully working to-do list using everything you learned.

HTML + JavaScript
<input id="taskInput" placeholder="Add a task...">
<button onclick="addTask()">Add</button>
<ul id="taskList"></ul>

<script>
let tasks = JSON.parse(localStorage.getItem("tasks")) || [];

function render() {
  const list = document.getElementById("taskList");
  list.innerHTML = tasks.map((t, i) =>
    `<li>${t} <button onclick="remove(${i})">✕</button></li>`
  ).join("");
}

function addTask() {
  const val = document.getElementById("taskInput").value.trim();
  if (!val) return;
  tasks.push(val);
  localStorage.setItem("tasks", JSON.stringify(tasks));
  render();
}

function remove(i) {
  tasks.splice(i, 1);
  localStorage.setItem("tasks", JSON.stringify(tasks));
  render();
}

render();
</script>
