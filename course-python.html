What is Python?

Python is one of the world's most popular programming languages. It's known for being simple, readable, and incredibly powerful — used in web development, data science, AI, automation, and more.

Why Python?

  • Easy to read — almost like plain English
  • Huge community and libraries
  • Used by Google, Netflix, NASA, and Instagram
  • Perfect for beginners AND professionals

Variables & Data Types

In Python, you don't need to declare a type — Python figures it out automatically.

Python
# String
name  = "Elham"

# Integer
age   = 22

# Float
gpa   = 3.8

# Boolean
enrolled = True

# Check the type
print(type(name))     # <class 'str'>
print(type(age))      # <class 'int'>

f-strings (the best way to format text)

Python
name = "Elham"
age  = 22

# Old way
print("My name is " + name + " and I am " + str(age))

# f-string (modern, preferred)
print(f"My name is {name} and I am {age}")
💡 Tip: Always use f-strings for combining variables and text in Python. They are cleaner, faster, and easier to read!

Loops

Loops let you repeat code without writing it multiple times.

for loop

Python
# Loop through a range
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# Loop through a list
fruits = ["apple", "banana", "mango"]
for fruit in fruits:
    print(fruit)

# Range with start, stop, step
for i in range(1, 10, 2):
    print(i)  # 1, 3, 5, 7, 9

while loop

Python
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1  # same as count = count + 1
💡 Tip: Be careful with while loops — always make sure the condition will eventually become False, or you'll create an infinite loop that freezes your program!

Lesson 5 of 10

Functions

Functions in Python are defined with the def keyword. They let you write reusable, organized code.

Python
# Define a function
def greet(name):
    return f"Hello, {name}!"

# Call it
print(greet("Elham"))  # Hello, Elham!

# Default parameter
def power(base, exp=2):
    return base ** exp

print(power(3))    # 9  (exp defaults to 2)
print(power(2, 10)) # 1024

# Multiple return values
def min_max(nums):
    return min(nums), max(nums)

low, high = min_max([3, 1, 9, 4])
print(low, high)  # 1 9
💡 Tip: Python functions can return multiple values at once — a feature unique to Python! The values are returned as a tuple and can be unpacked into separate variables.

Classes & OOP

Object-Oriented Programming (OOP) lets you model real-world things as objects with attributes (data) and methods (actions).

Python
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age  = age
        self.grades = []

    def add_grade(self, grade):
        self.grades.append(grade)

    def average(self):
        return sum(self.grades) / len(self.grades)

    def __str__(self):
        return f"{self.name}, age {self.age}"

# Create objects
s1 = Student("Elham", 22)
s1.add_grade(90)
s1.add_grade(85)
print(s1.average())  # 87.5
print(s1)            # Elham, age 22
💡 Tip: __init__ is the constructor — it runs automatically when you create an object. The self parameter refers to the object itself.

File Handling

Python makes reading and writing files very easy using the built-in open() function.

Python
# Write to a file
with open("notes.txt", "w") as f:
    f.write("Hello from Python!\n")
    f.write("Line two\n")

# Read from a file
with open("notes.txt", "r") as f:
    content = f.read()
    print(content)

# Read line by line
with open("notes.txt", "r") as f:
    for line in f:
        print(line.strip())

File Modes

  • "r" — read (default)
  • "w" — write (overwrites file)
  • "a" — append (adds to end)
  • "x" — create new file
💡 Tip: Always use the with statement when working with files. It automatically closes the file when you're done, even if an error occurs!

Build a Student Grade Calculator

Let's combine everything — a complete Python program that manages student grades.

Python
class GradeBook:
    def __init__(self):
        self.students = {}

    def add_student(self, name):
        self.students[name] = []
        print(f"Added student: {name}")

    def add_grade(self, name, grade):
        if name in self.students:
            self.students[name].append(grade)
        else:
            print(f"Student {name} not found!")

    def get_average(self, name):
        grades = self.students.get(name, [])
        if not grades:
            return 0
        return sum(grades) / len(grades)

    def report(self):
        print("\n=== Grade Report ===")
        for name, grades in self.students.items():
            avg = self.get_average(name)
            print(f"{name}: avg = {avg:.1f}")

# Use it
gb = GradeBook()
gb.add_student("Elham")
gb.add_grade("Elham", 90)
gb.add_grade("Elham", 85)
gb.add_grade("Elham", 92)
gb.report()
# Elham: avg = 89.0
