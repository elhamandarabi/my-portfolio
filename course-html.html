What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. Every website you visit is built using HTML.

What does HTML do?

HTML tells the browser what to show — headings, paragraphs, images, links, and more. Think of it as the skeleton of a webpage.

  • HTML uses tags to wrap content
  • Tags usually come in pairs: an opening tag and a closing tag
  • A complete HTML file has a specific structure

Your first HTML page

HTML
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
💡 Tip: Every HTML file starts with <!DOCTYPE html> — this tells the browser which version of HTML you're using (HTML5).

Key HTML Tags

  • <html> — The root element of the page
  • <head> — Contains meta info (title, links to CSS)
  • <body> — Contains everything visible on the page
  • <h1> to <h6> — Headings (h1 is largest)
  • <p> — Paragraph of text

HTML Links, Images & Lists

Now that you know the basics, let's look at some essential HTML elements you'll use in every webpage.

Links — <a>

Links connect pages together. The href attribute tells the browser where to go.

HTML
<a href="https://google.com">Visit Google</a>
<a href="about.html" target="_blank">Open in new tab</a>

Images — <img>

HTML
<img src="photo.jpg" alt="A description" width="300">
💡 Tip: Always add an alt attribute to images. It helps screen readers and improves accessibility.

Lists

HTML
<!-- Unordered (bullets) -->
<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<!-- Ordered (numbers) -->
<ol>
  <li>Step one</li>
  <li>Step two</li>
</ol>

Introduction to CSS

CSS (Cascading Style Sheets) controls how HTML elements look — colors, fonts, sizes, spacing, and layout.

How to add CSS

HTML + CSS
<!-- In the <head> of your HTML -->
<style>
  h1 {
    color: blue;
    font-size: 32px;
  }
  p {
    color: #666;
    line-height: 1.6;
  }
</style>

CSS Selectors

  • Tag selector — targets all elements of that type: p { }
  • Class selector — targets elements with a class: .myClass { }
  • ID selector — targets one specific element: #myId { }
CSS
/* Class selector */
.highlight {
  background: yellow;
  padding: 5px;
}

/* ID selector */
#header {
  background: #2563eb;
  color: white;
}
💡 Tip: Use classes (.) for styles you'll reuse on multiple elements. Use IDs (#) for unique elements like a navigation bar or footer.

The CSS Box Model

Every HTML element is a rectangular box. The box model describes the space that element takes up — content, padding, border, and margin.

Box Model Layers

  • Content — The actual text or image inside the element
  • Padding — Space between content and the border (inside)
  • Border — A line around the padding
  • Margin — Space outside the border (between elements)
CSS
.card {
  width: 300px;
  padding: 20px;        /* inner space */
  border: 2px solid #ccc; /* border line */
  margin: 16px;        /* outer space */
  border-radius: 8px; /* rounded corners */
}
💡 Tip: Use box-sizing: border-box; in your CSS reset to make width calculations simpler. This way padding and border are included in the total width.

Display Property

The display property controls how an element behaves in the layout:

  • block — takes full width, starts on a new line (div, p, h1)
  • inline — only takes up as much width as needed (span, a)
  • flex — powerful layout system for rows/columns

CSS Flexbox

Flexbox is a CSS layout system that makes it easy to align and distribute elements in a row or column.

Basic Flexbox

CSS
.container {
  display: flex;
  justify-content: space-between; /* horizontal align */
  align-items: center;          /* vertical align */
  gap: 16px;                    /* space between items */
}

justify-content values

  • flex-start — items align to the left
  • flex-end — items align to the right
  • center — items are centered
  • space-between — equal space between items
  • space-around — equal space around items

Flex Direction

CSS
.column-layout {
  display: flex;
  flex-direction: column; /* stack vertically */
  gap: 12px;
}
💡 Tip: Flexbox is one of the most used CSS features. Master it and you can build almost any layout you can imagine!

Responsive Design & Media Queries

A responsive website looks good on all screen sizes — phone, tablet, and desktop. CSS Media Queries let you apply different styles based on the screen width.

CSS
/* Default (desktop) styles */
.container {
  display: flex;
  gap: 2rem;
}

/* Tablet and below */
@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

/* Mobile only */
@media (max-width: 480px) {
  h1 {
    font-size: 24px;
  }
}
💡 Tip: Always add <meta name="viewport" content="width=device-width, initial-scale=1.0"> in your HTML head — without it, media queries won't work correctly on mobile.

Common Breakpoints

  • 480px — Small phones
  • 768px — Tablets
  • 1024px — Laptops
  • 1200px+ — Large desktops

HTML Forms

Forms let users enter data — login pages, contact forms, search bars. The <form> element wraps all input elements.

HTML
<form action="/submit" method="post">

  <label for="name">Your Name</label>
  <input type="text" id="name" placeholder="John">

  <label for="email">Email</label>
  <input type="email" id="email" required>

  <textarea rows="4" placeholder="Your message..."></textarea>

  <button type="submit">Send</button>
</form>

Common Input Types

  • text — single line text
  • email — validates email format
  • password — hides characters
  • number — only numbers allowed
  • checkbox — tick box
  • radio — pick one from a group

Lesson 8 of 10

CSS Grid

CSS Grid is a two-dimensional layout system — it handles both rows and columns at the same time. It's perfect for page layouts and card grids.

CSS
.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* 3 equal columns */
  gap: 20px;
}

/* Auto-fill: as many columns as fit */
.auto-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: 16px;
}
💡 Tip: repeat(auto-fill, minmax(250px, 1fr)) is a magic one-liner — it creates as many columns as the screen allows, each at least 250px wide. Great for responsive card grids!

Grid vs Flexbox

  • Use Flexbox for 1D layouts (single row or column)
  • Use Grid for 2D layouts (rows AND columns)

CSS Variables & Transitions

CSS Variables (custom properties) let you store values you reuse across your stylesheet. Change it in one place and it updates everywhere.

CSS
/* Define variables */
:root {
  --primary: #2563eb;
  --text: #0f172a;
  --radius: 8px;
}

/* Use variables */
.button {
  background: var(--primary);
  border-radius: var(--radius);
  color: white;
}

CSS Transitions

Transitions make property changes animate smoothly instead of jumping instantly.

CSS
.button {
  background: #2563eb;
  transition: background 0.3s, transform 0.2s;
}

.button:hover {
  background: #1d4ed8;
  transform: translateY(-2px); /* lift up */
}
💡 Tip: Transitions make your website feel polished and professional. Even a simple 0.2s transition on hover makes a huge difference!

Build a Complete Web Page

In this final lesson, we bring everything together. Here's a complete mini webpage using everything you've learned.

HTML + CSS
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>My Portfolio</title>
  <style>
    :root { --blue: #2563eb; }
    body { font-family: sans-serif; margin: 0; }
    nav { background: var(--blue); color: white; padding: 1rem 2rem; display: flex; justify-content: space-between; }
    .hero { text-align: center; padding: 4rem 2rem; }
    .cards { display: grid; grid-template-columns: repeat(auto-fill, minmax(200px,1fr)); gap: 1rem; padding: 2rem; }
    .card { border: 1px solid #ddd; border-radius: 8px; padding: 1.5rem; transition: transform 0.2s; }
    .card:hover { transform: translateY(-4px); }
    @media (max-width: 600px) { .hero h1 { font-size: 24px; } }
  </style>
</head>
<body>
  <nav><span>MyPortfolio</span><a href="#" style="color:white">Contact</a></nav>
  <div class="hero"><h1>Welcome to My Site</h1><p>I build cool things.</p></div>
  <div class="cards">
    <div class="card"><h3>Project 1</h3><p>Description here.</p></div>
    <div class="card"><h3>Project 2</h3><p>Description here.</p></div>
  </div>
</body>
</html>
